## 项目背景
传统的线下拍卖存在场地限制、参与门槛高、信息不透明等问题。本系统旨在开发一个基于移动端的在线拍卖平台，支持卖家发布拍品、买家实时竞价、管理员审核管理，并引入 AI 技术辅助商品描述生成和智能估价。

## 技术栈
前端：微信小程序（买家/卖家）+ Vue3 + Element Plus（管理员）
后端：Node.js + Express + TypeScript + Prisma 7
数据库：PostgreSQL 15 + pgvector + Redis 7
AI：DeepSeek V4 Flash + BAAI/bge-m3 + Tavily 联网搜索

## 核心功能
1. 四角色（卖家/买家/拍卖管理员/系统管理员）权限系统
2. 拍品 CRUD + 图片上传 + 管理员审核
3. 拍卖场次管理 + 定时器（10s）自动开拍/结拍
4. WebSocket 实时竞价 + HTTP 双通道
5. 订单生命周期：支付→发货→确认收货
6. 争议处理：发起→受理→解决
7. AI 描述优化（必选）：名称+成色+原价 → DeepSeek 生成卖点文案
8. AI 智能估价（加分）：bge-m3 向量化 → pgvector 检索同类成交 → Tavily 联网搜索 → DeepSeek 流式 SSE 输出估价建议
